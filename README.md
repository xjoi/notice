## 机房守则  
#### 1. 非休息时间严格禁止玩游戏、看视频、看网络小说等与竞赛无关的活动；  
#### 2. 比赛训练独立完成，严禁上网搜索试题、相关材料、看书、讨论交流；  

#### 凡违反以上条例者，剥夺机房使用权3天。多次违反或情节严重者加大处罚，直至永久禁止参与训练。
